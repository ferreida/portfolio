Mes projets - Projet sur l'instalation d'un poste de travail

Durant ce projet, nous devions faire une instalation complète de machine virtuelle. Pour ce faire, nous devions dans un premier temps comparer à l'oeil nue les images ISO, celle contenue sur un site et celle obtenue à partir d'une certaines commandes dans l'invite de commande. Ensuite, nous devions respecter certaines contraintes tels que la langue de la machine virtuelle et du clavier, nous pouvions créer un utilisateur pour accéder à cette machine virtuelle afin de pouvoir l'utiliser.

Une fois l'installation finie, nous devions nous connecter en tant qu'utilisateur root afin d'installer des applications tel que gedit.

Cette image représente ma carte mentale que j'avais fait pour la SAE S1.03 expliquant comment installer une machine virtuelle sur un ordinateur linux.

Nous devions faire cette carte mentale afin de montrer aux personnes ne s'y connaissant pas en informatique comment faire pour installer et utiliser une machine virtuelle.
