Mes projets - Projet base de données

Lors de ce projet, nous devions faire une base de données concernant les diverses informations du Titanic que, nous avions pu trouver lors de nos recherches. Nous devions faire un petit compte rendu de la manière dons le nauffrage du Titanic à eu lieu, c'est-à-dire comment l'équipage s'y est pris pour gérer les passagers ainsi que comment plus de personnes auraient pu être sauvé.

Par la suite, nous devions faire le schéma relationnel de la base de données afin de la faire valider par nos enseigants avant de pouvoir construire notre base de données.

Ensuite nous devions créer notre base de données en respectant les contraintes relationnelles imposées par le sujet.

Une fois cette base de données créée nous devions faire plusieurs requêtes sur notre base afin de vérifier que nous avions bien respecté les contraintes et que nos données sont correctes.
